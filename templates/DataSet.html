Data Acquisition

The Data are obtained from Trulia API and Trulia house search website. And I scraped over 400 pages of trulia house search website and acquired more than 5000 housing properties. After the elimination of the housing properties which don't have key information(housing area and housing prices). I eventually have 2514 housing properties at my hand.

Data Description

Each housing item has the following features:

Latitude, Longitude, Address, Postal_code, Housing_type, #Baths, #Beds, Crime_level,Built year, Housing_price/SQFT
